## 📂 文件结构

config 下的文件会在适当的时候自动加载，所以你不需要手动引入这些文件。
**WhimsCFGS** 带有一组默认配置文件，这些文件将在您的配置**之前**加载。

<pre>
~/CFGS/
├── config
│   ├── autoexec.cfg
│   ├── keymaps.cfg
│   └── options.cfg
│  
├── plugins
│     ├── spec1.cfg
│     ├── **
│     └── spec2.cfg
└───init.cfg
</pre>
